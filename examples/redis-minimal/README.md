## Getting Started

First, run the development server:

```bash
npm i
npm run dev
```

Or run build and run production server:

```bash
npm i
npm run build
npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Modify `.env` file if you need to.
